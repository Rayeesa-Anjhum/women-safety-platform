# 🛡️ SafeGuard - Women Safety & Emergency Assistance Platform 🚀

SafeGuard is a real-time, responsive Full-Stack web application built to provide immediate assistance to women in emergency situations. With a single tap, the platform captures the user's live GPS coordinates and broadcasts them instantly to a central command center and configured emergency contacts.

## ✨ Key Features
* **🚨 One-Tap Instant SOS:** A high-visibility, 3D animated emergency panic button that triggers immediate location sharing upon clicking.
* **📍 Live GPS Tracking Map:** Automatically fetches precise coordinates via the HTML5 Geolocation API and displays the live position on an interactive dark-themed Leaflet.js map.
* **👤 Emergency Profile Management:** A smooth, glassmorphism-designed registration dashboard allowing users to securely save their critical contact details.
* **⚡ Real-Time Responder Insights:** Simulates network callbacks to keep users updated on nearby active safety teams and response status.

## 🛠️ Tech Stack
* **Frontend:** HTML5, JavaScript (ES6+), Tailwind CSS (Frosted Glassmorphism UI)
* **Backend:** Node.js, Express.js
* **Mapping API:** Leaflet.js
* **Database / Storage:** Volatile Array Storage / JSON Local Storage (Ideal for POC and lightweight integration)
